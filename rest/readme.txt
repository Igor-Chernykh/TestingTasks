Есть белая страница (x,y), на которой можно рисовать многоугольники (мышкой или как-то иначе). При этом, можно менять ранее сохраненную фигуру. Ограничений по цвету нет. Ограничение по количеству углов многоугольника нет. Ограничение по расположению фигур нет (могут накладываться друг на друга в рамках страницы).
Задание:
1. Какие данные необходимо сохранять в БД? Нужно описать (и обозвать) все данные которые нужно хранить для удовлетворения задачи
2. Какие методы REST API необходимы для взаимодействия frontend с backend (нужно просто описание методов без синтаксиса)? Нужно просто словесно описать состав методов и их структуру.
3. Задача со звёздочкой. Фигуры не могут накладываться.


Необходимо разработать простенький документ, в котором следует описать структуру и связи таблиц БД. Методы и структуру запросов.
